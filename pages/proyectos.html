Android (Kotlin)

En primer lugar tenemos mi último proyecto profesional. Esta App para Android fue creada en verano de 2024 para el restaurante El Tinglado de Lagunillas, en Málaga. La aplicación era un proyecto a medida, que sirve para gestionar las reservas de los clientes. Se caracteriza por su sencillez en el uso, y su rápidez a la hora de acceder a los datos. En este caso, se utiliza una base de datos en tiempo real como Firebase junto con el lenguaje Kotlin.

Html5, Css y Javascript

De este proyecto estoy especialmente orgulloso. Esta web la realice cuando empecé a darme cuenta que mis padres tenían problemas de memoria. La mecanica tenía que ser muy simple para que unas personas mayores pudiesen jugar intuitivamente. En este caso, en la pantalla hay cuatro cuadrados grandes de distintos colores: azul, verde, amarillo y rojo. El programa resalta estos cuadrados aleatoriamente, generando una secuencia de colores que hay que repetir. Cada vez que se acierta una secuencia, se aumenta la dificultad, añadiendo más colores a la secuencia. Se puede controlar con el ratón, pero es preferible usar un mando de Xbox, ya que los botones tienen los mismos colores que los cuadrados de la pantalla. Además, los gatillos del mando sirven de ayuda, ya que con el izquierdo se repite la secuencia y con el derecho aparecen unos cuadraditos en la parte inferior con la secuencia a realizar. No solo mantuve un tiempo ocupados a mis padres si no que también mejoraron un poquito su memoria.

Arduino (electrónica y programación en C)

Esta es la versión de Arduino que hice del proyecto anterior. En este caso la mecanica del juego cambiaba: cada vez que se acertaba una secuencia de colores, la siguiente combinación se mostraba más rápido.

Emulador de Chip 8 en C

Mis primeros pasos en C fue hacer este emulador de Chip-8 con el que se podían jugar a varios juegos. Fue muy interesante y divertido, la mayoría de cosas las aprendí de tutoriales de Youtube, como Makigas, pero también tuve que buscar mucha información en Stackoverflow. En el video el emulador carga una versión del mítico Pong.

Aplicación de escritorio en C#: Deutschapp

Esta aplicación la cree mientras estudiaba Alemán en la Escuela de Idiomas de Málaga. La hice para ayudarme a memorizar sustantivos, verbos, declinaciones, etc... Mi idea era compartirla con mis compañeros de clase, para que pudieramos crear entre todos la base de datos. Está hecha en C#.

Deutschapp Web: Php y MySQL

La aplicación en C# de Deutschapp estaba muy bien, pero tenía un problema. Estaba compilada para Windows y algunos de mis compañeros tenían Mac. Solución: Hacer una versión web y así funcionaría en sistemas Windows, Mac, Android y Iphone. Personalmente me fue de mucha utilidad para repasar las declinaciones y mejoré bastante mi nivel.

Web gestor de tests: Php y MySQL

Un tiempo después de hacer la web de Deutschapp, la idea les gustó tanto a dos amigos que me propusieron llevarla más allá y usarla para crear test para opositores. En concreto, la web se usó para preparar oposiciones para bomberos y para Capacitación de transporte por carretera. Ahora ya había gestión de usuarios, los cuales podían crear sus propios tests o inscribirse en otros que compartian otros usuarios.

Videojuegos: Unity (C#)

Durante un periodo de tiempo, me plantee desarrollar videojuegos, así que aprendí a usar Blender para modelar en 3D y Unity, para crearlos. Programaba en C#. Por desgracia no llegué muy lejos, pero aprendí un montón y me diverti bastante. El primer video es un prototipo para un sistema de ataque con armas controlado con el ratón. El segundo video fue una prueba de animación facial con la voz de Constantino Romero. En aquellos tiempos era bastante dificil animar facialmente un personaje y el cuerpo, así que me tuve que inventar un truco. El resultado quedó bastante bien.
